To run program:

1. Compile program by typing to terminal:
        gcc main.c commands.c -o smallsh --std=gnu99

2. To run program:
        ./smallsh

3. To run program with test script:
        place ./p3testscript in same directory as compiled shell
        
        chmod it (chmod +x ./p3testscript)

        Run program with test script:
                ./p3testscript 2>&1
